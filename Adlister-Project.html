Being introduced to the Spring framework after this project was mind blowing.

Our first full-stack application with CRUD functionality!

Texas Adlister

This was the first project at Codeup where we worked in groups of three; this allowed us to experience the paired partner programming dynamic in a much bigger way. During our first day we spent it entirely on database design, user experience, and mapping out how we can use Java to work with MySQL. Even though we spent plenty of time planning; when it came time to code there were many hurdles that we had to jump through together. Luckily we were able survive all three of us working on the project simultaneously and merging to Master with no Github conflicts!

  • The Tasks
  • Use Java to interact with the web, and build a full-stack web application.
  • Learn the Model-View-Controller Paradigm.
  • Understand the request-response lifecycle of HTTP requests.
  • Be able to use JSPs and the JSTL to generate html pages.
  • Implement fully functional full-stack application with CRUD basic functions.

After days of struggling to figure out how to CRUD with a database...

  • The Outcome
  • Established database connections and used connections to execute queries.
  • Created servlets that are mapped to a URI.
  • Used prepared statements to avoid SQL injection.
  • Implemented features for a user to create an account, login, create an ad, update, and delete.
  • Used Java to interact with a MySQL database.

Project Demo

Even though we finished our project as others were presenting and using every minute to plan our demonstration; we were very happy with how everything turned out. Alexander Lee and Miguel Garcia were pleasant to work with and we all were able to contribute equally to the success of our first full-stack application.
