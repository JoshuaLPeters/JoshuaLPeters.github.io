Capstone Project

Contagion Watch is a full-stack application developed in SPRING boot to help you discover, learn, and track deadly diseases. For the scope of this project, you can discover where these major diseases are in the world: Coronavirus, Ebola, and Malaria. It is possible to view statistics from over 200 countries in our database. News articles and dynamic maps allow you to also track these outbreaks as they happen in real time, keeping you up to date with the latest information.

Dynamic Map

For the development process, our team used AGILE workflow to keep track of every step towards completion. My responsibilities for our application included designing the user interface and user experience of the website. Through the use of the Bootstrap framework and multiple JavaScript styling libraries, I wanted to convey the sense of urgency that comes with tracking these diseases. Before this project began we identified a problem, the information about the Coronavirus was out there, but it wasn’t represented dynamically. Our solution was to develop a dynamic map that updates with every entry, and to include additional information about that disease.

We are able to CRUD disease entries with our map!

Featured Functionality

One can also find out the statistics for any country that they are curious about. Let’s say that you want to know how much of the Corona outbreak is currently affecting the United States. The locations page lists all the countries that are affected by the diseases. We can then use the search filter to find our country, and then its page displays the data here which is divided into each disease. For the admin running this application there is also functionality to create, edit, and delete!
