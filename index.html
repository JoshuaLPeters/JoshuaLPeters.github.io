About Me

Hello world! My non-traditional path in life has recently taken me through Codeup's web development bootcamp program! I will be graduating at the end of February and will be starting a new career in the tech industry.

For the past five years I've been running my own small business as a Partnered Twitch Broadcaster which started as a hobby when I was about to deploy with the Air Force. This past decade has been a wild ride between the military and the entertainment industry but I am very excited to transition into the world of coding in 2020 as a software developer with all the training Codeup has provided.

Software Development

My coding adventure began mid-August starting with the pre-work required to attend Codeup's web development course! The past several months have been crammed with HTML, CSS, JavaScript, jQuery, Java, and projects with my piers from the Deimos Cohort.

The instructors enabled each of us to accelerate our learning by providing a clear and concise cirriculum on what the industry wants today from software developers applying for jobs. Here are some of the projects recently completed at Codeup:

Coffee Project

The Coffee Shop project was designed to teach us how to build a site that was interactive with our javascript. This exercise was built with River Robins; which was another great experience getting to work with a partner to solve a group of problems together. We needed to resolve:

  • Update HTML so the website is presentable
  • Read code and understand the existing structure
  • Refactor code to properly display coffees
  • Add functionality to navigate and display coffees
  • Use paired programming development technique

After using our Code Kung-Fu and being patient with each other...

  • Developed search capabilities to interact with website
  • Iterated through data structures; primarily arrays
  • Mastered the forEach loop
  • Used DOM to dynamically update the page
  • Designed a professional product using Bootstrap

Overall the project with River Robins went very well once we understood the code that needed to be refactored. This was our first project where the code was pre-built so the greatest challenge was learning how to use what we started with to resolve the project requirements.

Movie Project

The Movie Project was the third major project that involved the paired partner programming technique. This project emphasized on learning how to utilize npm to build out a working movie site that could search, add, edit, and delete data pulled from existing JSON files or an API.

  • Learn how to uilitze npm to build a program
  • Use promises with javascript
  • Plan a detailed step by step process
  • Make ajax requests to pull movies from an API
  • Allow users to add, edit, delete, and search movies

After being defeated by many .then's and finding lost curly brackets that we accidentally deleted...

  • Initialized JSON database to load into the site with ajax
  • Created search function for movie titles with API
  • On page load generated HTML from JSON responses
  • Iterated through a complex API to retrieve specific info
  • Merged to Github while rotating with paired programming

Working with Alexander Lee made navigating the MovieDB API tolerable to say the least; his inept ability to work through these complex objects made iterating through them much easier to find the desired information to complete this project in a timely manner. Our greatest challenge was solving a syntax error towards the end when we had accidentally deleted a pair of brackets that wasn't picked up by IntelliJ. It was a relieving moment with huge smiles once our code was working to the desired specifications.
